# Nimbus Frontend

Landing page profesional para Nimbus - Sistema de Alerta Temprana Meteorológica con IA para San Cristóbal, Galápagos.

## 🚀 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Previsualizar build de producción
npm run preview
```

## 🎨 Características

- **Diseño Moderno**: Interfaz limpia y profesional con animaciones fluidas
- **Dark Mode**: Modo oscuro con toggle persistente
- **Responsive**: Completamente adaptable a todos los dispositivos
- **Performance**: Optimizado con React 18 + Vite
- **Animaciones**: Framer Motion para transiciones suaves
- **Iconos**: Lucide React para iconografía profesional

## 🎨 Paleta de Colores

## 🌐 Secciones

1. **Hero**: Presentación principal con stats y CTAs
2. **Problem**: El desafío que resuelve Nimbus
3. **Solution**: Cómo funciona el sistema
4. **Audiences**: Tres interfaces para tres tipos de usuarios
5. **Technology**: Stack técnico y arquitecturas de IA
6. **Impact**: Roadmap y escalabilidad
7. **Footer**: Links y contacto

## 📝 Notas

- Las imágenes deben estar en `/Public/`
- El modo oscuro se persiste en localStorage
- Todas las secciones tienen animaciones on-scroll
- Navegación smooth scroll con anclas
